# Cake House - Artisanal Pâtissier

This is a premium, high-conversion bakery website for a Mysore-based artisanal pâtissier.

## Features
- Mobile-first, premium user experience.
- Seamless cake browsing and ordering.
- WhatsApp integration for order checkout.
- Optimized for quick user conversion (<30 seconds).

## Tech Stack
- [Next.js](https://nextjs.org/)
- Tailwind CSS
- Framer Motion

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
